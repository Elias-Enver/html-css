Usando Links Internos

Esta é a primeira pagina do site se quiser minha segunda pagina

Usa-se rel=next para referenciar a proxima pagina de link interno e rel=prev para a pagina anterior.

O rel=nofollow serve para links externos mas você não da aval para esse conteudo.

O Target="_self" ou sem target abre na mesma pagina.
